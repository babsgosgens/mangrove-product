So - what's a package manager?

If you have used a computer or similar device in the past couple of years, you will certainly have used a package manager - whether you know it or not.

You have probably used it to install software via an app store or to update existing software on your computer with an automatic updater.

mangrove brings that concept into the joomla world in a more elegant way than existing solutions like the the joomla updater.
